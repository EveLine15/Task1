📜 История веб-технологий

HTML (HyperText Markup Language)

  • 1989 - Тим Бернерс-Ли предложил концепцию.
  • 1993 - Первая версия HTML.
  • 1995 - HTML 2.0 (стандарт IETF).
  • 2014 - HTML5 стал рекомендацией W3C.

CSS (Cascading Style Sheets)

  1. 1994 - Хокон Виум Ли предложил CSS.
  2. 1996 - CSS1 официально выпущен.
  3. 1998 - CSS2 добавлены медиа-типы.
  4. 2011-2021 - CSS3 модульный подход.

JavaScript

1995
Брендан Эйх создал Mocha (позже LiveScript, затем JavaScript) за 10 дней.
1997
ECMAScript 1 — стандартизация.
2009
ECMAScript 5 (ES5) — эра современного JS.
2015+
Ежегодные обновления ES6, ES7 и т.д.
